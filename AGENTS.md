# Agents

## End of Session Checklist

At the end of every work session, always perform these steps in order:

1. **Build** — Run a full project build and fix any errors or warnings.
2. **Test** — Run all tests and fix any failures before proceeding.
3. **Changelog** — Append a dated entry to `CHANGELOG.md` summarizing what was done in the session.
4. **Commit** — Stage all changes and create a git commit with a clear, descriptive message.
